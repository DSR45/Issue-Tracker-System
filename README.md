# Mini Issue Tracker
I built this project as part of my backend learning journey. This is my first project I built using spring boot and React js. 

## Features
- Create Issue
- Create User
- Assign Issue to User
- Track the status of Issue [ Resolved | Open ]
- Arrange Issues based on status

## Tech Stack
- Spring boot
- Reactjs
- mysql
- REST API's

## How it's work
 1. User load the website.
 2. Get the data from backend server.
 3. Shows all the issues, Resolved and Open both.
 4. On create or remove action, Issue adds into the database and quickly display on screen.
 5. Assign the issue to user by clicking user icon at the bottom.
 6. Track your status.
